attribute selector

type of attribute selector

  • descent selector
  • child selector
  • sibling selector

descent selector:
the descent selector matches all elements that are descent of specific selector.

child selector:
the child selector select all elements that are direct child of specific element.

general sibling selector:
the general sibling selector selects all element that arenext sibling of a specified element

adjecent-siblind selector: the adjecent-siblind selector is used to select an element that is directly after the specified element.
